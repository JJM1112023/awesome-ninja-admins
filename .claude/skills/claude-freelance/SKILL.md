# Claude Code Freelance: $5K Local Business Sites

## The core insight

Claude Code doesn't just help you build faster — it restructures what you can charge.

A website that used to require a senior developer and 2–3 weeks of work now takes 2 hours. The client still pays for the **outcome** (a professional website), not the hours. Your cost went to near-zero. Your price didn't have to.

```
Before Claude Code:   40 hrs work × $75/hr = $3,000 (underpaid and exhausted)
After Claude Code:     2 hrs work × $2,500/hr effective rate = $5,000 (same client, same invoice)
```

## Who buys this

**Local businesses** are the best-fit client for this model:
- Roofing, HVAC, plumbers, landscapers, dentists, restaurants, law firms
- They need: professional presence, Google-rankable pages, mobile-ready site, contact forms
- They don't need: custom SaaS, complex integrations, app stores
- They pay: $2,000–$8,000 for a site they'd previously wait months to get

**The pitch:**
> "I build custom websites for [industry] businesses. Most agencies charge $5–10K and take 6–8 weeks. I deliver in 5–7 days."

You're not competing on price — you're competing on **speed**.

## The WordPress + Claude Code workflow

WordPress is the right stack for local business clients because:
- Clients know it and can log in and read content
- Hosts (Hostinger, SiteGround) set it up in one click
- SEO plugins (Yoast, RankMath) are already built
- Maintenance plans ($99–$299/mo) are easy to upsell

### Build sequence

1. **Gather the brief** (30 min with client or async form)
   - Business name, location, services, target customer
   - 3 competitor sites they like
   - Logo, photos (or use AI-generated placeholders)
   - Primary goal: calls, form fills, bookings

2. **Extract a blueprint** (15 min)
   - Find a reference site on Dribbble or Awwwards that fits the industry
   - Use Firecrawl to extract the layout structure
   - Or: screenshot + describe to Claude

3. **One-shot build in Claude Code** (60–90 min)
   ```
   Build a responsive WordPress-ready HTML/CSS site for a [industry] business.

   CLIENT: [name], [city], serves [target customer]
   SERVICES: [list]
   TONE: [professional / friendly / premium]
   REFERENCE: [describe or attach blueprint]
   PRIMARY CTA: [call now / book online / get a quote]

   DELIVER:
   - index.html with hero, services, about, testimonials, contact, footer
   - Embedded CSS (single file, no dependencies)
   - Mobile responsive
   - Schema markup for local SEO (LocalBusiness, services)
   - Contact form (mailto: for now, swap to Gravity Forms on WP)
   ```

4. **Polish pass** (20 min) — spacing, mobile check, CTA visibility

5. **Deploy to WordPress** (20 min)
   - Export HTML → convert to WP theme or use Elementor to rebuild
   - Or: host as static on Vercel, point domain → instant live

6. **Deliver + upsell** — hand off, offer maintenance plan

## The economics of local business SEO sites

Jesse's model (Rank Expand) layers SEO on top of the build:

| Service | Price | Delivery |
|---------|-------|----------|
| Website build | $3,000–$5,000 | 1 week |
| SEO setup (on-page + local) | $500–$1,500 | included |
| Monthly SEO maintenance | $500–$2,000/mo | recurring |
| Google Business Profile optimization | $300–$500 | one-time |

The website is the entry point. The monthly retainer is the business.

**Target:** 10 clients × $500/mo = $5,000/mo recurring before any new builds.

## What to say to close the deal

**Cold outreach angle:**
> "I searched [city] + [service] on Google and noticed your site isn't ranking. I help [industry] businesses get found by customers in [city]. Want me to show you what I'd build for you?"

**Social proof shortcut:**
Build 2–3 spec sites for fake local businesses (or friends' businesses for free). Use them as your portfolio. Show the speed: "I built this in an afternoon."

**Objection: "We already have a website"**
> "I can see it. It's not ranking for [keyword]. The site I build will be optimized for that from day one."

## Scaling with systems

Once the workflow is repeatable, systematize it:

```
1. Onboarding form → auto-fills the build brief
2. Claude Code session → site generated
3. Review pass → polish
4. Vercel/WordPress deploy → live
5. Handoff email → template with login credentials + next steps
6. 30-day follow-up → maintenance plan close
```

Target: 2–3 builds per week = $6K–$15K/week at full capacity.
